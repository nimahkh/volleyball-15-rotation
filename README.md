# Volleyball 5-1 Rotation Helper

Interactive visualization of 5-1 rotations with lineup input, serve/receive/base views, and simple rotation controls.

## Features
- Lineup entry for 7 roles (OH1, OH2, Opp, Setter, MB1, MB2, Libero).
- Receive, Serve, and Base (rotation) views.
- Animated receive sequence to illustrate transitions.

## Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS

## Getting Started
1. Install dependencies: `npm install`
2. Start the dev server: `npm run dev`
3. Build for production: `npm run build`
4. Preview the production build: `npm run preview`

## Linting and Type Checking
- Lint: `npm run lint`
- Type check: `npm run typecheck`
